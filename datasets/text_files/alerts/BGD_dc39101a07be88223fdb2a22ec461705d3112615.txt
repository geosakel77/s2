Impact: According to VideoLan’s security bulletin, this vulnerability can be exploited by creating a specially crafted file and tricking a user into opening it with VLC. If successful, a malicious third party could trigger either a crash of VLC or an arbitratry code execution with the privileges of the target user. Mitigations: Due to the severity of this vulnerability it is strongly advised that all users download and install version 3.0.11.
