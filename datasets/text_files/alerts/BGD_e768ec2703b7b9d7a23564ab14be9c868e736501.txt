This could result in a Denial of Service attack. There are no such structures used within SSL/TLS that come from untrusted sources so this is considered safe Impact : An attacker can exploit this issue to cause denial-of-service conditions. OpenSSL 1.1.0 users should upgrade to 1.1.0h OpenSSL 1.0.2 users should upgrade to 1.0.2o Mitigation : Updates are available. Please see the references or vendor advisory for more information.
