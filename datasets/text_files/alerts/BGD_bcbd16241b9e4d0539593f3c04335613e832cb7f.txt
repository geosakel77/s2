An attacker could exploit this vulnerability by posting a crafted request to the user interface of Cisco UCS Central. Impact : Command Execution Vulnerability Vulnerable Products This vulnerability affects Cisco UCS Central Software prior to Release 2.0(1c). Mitigation : Cisco has released software updates that address this vulnerability. Please see the references or vendor advisory for more information.
