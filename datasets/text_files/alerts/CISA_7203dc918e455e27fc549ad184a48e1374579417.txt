More Alerts Alert (SA07-109A) Apple Updates for Multiple Vulnerabilities Original release date: April 19, 2007 Systems Affected Apple Mac OS X version 10.3.x (Panther) and version 10.4.x (Tiger) These vulnerabilities affect both Intel-based and PowerPC-based Apple systems. Overview Apple Mac OS X is affected by multiple vulnerabilities. Apple has released Security Update 2007-004 to address these vulnerabilities, the most serious of which may allow a remote attacker to place and run malicious programs on your computer. Solution Install an Update Install Apple Security Update 2007-004 through Apple Update . Description Apple Mac OS X is affected by multiple vulnerabilities. Some of these vulnerabilities could allow an attacker to run malicious programs on your computer. For more technical information, see US-CERT Technical Alert TA07-109A .
