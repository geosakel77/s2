Impact This vulnerability allows for unauthenticated attackers, or authenticated users, with network access to the TMUI, through the BIG-IP management port and/or Self IPs, to execute arbitrary system commands, create or delete files, disable services, and/or execute arbitrary Java code. This vulnerability may result in complete system compromise. The BIG-IP system in Appliance mode is also vulnerable. Vulnerability CVE-2020-5902 received a CVSS score of 10(most severe score). Mitigation F5 recommends upgrading to a fixed software version to fully mitigate this vulnerability(CVE-2020-5902).
