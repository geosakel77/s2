Researchers at the MS Exchange Server team have released a script that could be used by administrators to check if their installs are vulnerable to the recently disclosed vulnerabilities. Microsoft released the tool as open-source on GitHub, it can be used to check the status of Exchange servers. “Formerly known as Test-Hafnium, this script automates all four of the commands found in the Hafnium blog post.” states Microsoft. “It also has a progress bar and some performance tweaks to make the CVE-2021-26855 test run much faster. Download the latest release here: Download Test-ProxyLogon.ps1( https://github.com/microsoft/CSS-Exchange/tree/main/Security ).
