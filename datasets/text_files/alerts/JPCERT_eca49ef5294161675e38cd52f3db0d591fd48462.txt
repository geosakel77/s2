I. Overview

  Microsoft has released its security bulletin for July, 2015.
This bulletin contains four (4) updates that are rated as "critical".
Remote attackers leveraging these vulnerabilities may be able to
execute arbitrary code.

  Details on the vulnerabilities can be found at the following URL:

    Microsoft Security Bulletin Summary for July 2015
    https://technet.microsoft.com/en-us/library/security/ms15-jul

  [Security updates rated as "critical"]

    MS15-065
    Security Update for Internet Explorer (3076321)
    https://technet.microsoft.com/en-us/library/security/ms15-065

    MS15-066
    Vulnerability in VBScript Scripting Engine Could Allow Remote Code Execution (3072604)
    https://technet.microsoft.com/en-us/library/security/ms15-066

    MS15-067
    Vulnerability in RDP Could Allow Remote Code Execution (3073094)
    https://technet.microsoft.com/en-us/library/security/ms15-067

    MS15-068
    Vulnerabilities in Windows Hyper-V Could Allow Remote Code Execution (3072000)
    https://technet.microsoft.com/en-us/library/security/ms15-068

  According to Microsoft, attacks leveraging MS15-065 (critical) have
been observed in the wild.


II. Solution

  Please apply the security update programs through Microsoft Update,
Windows Update, etc. as soon as possible.

    Microsoft Update
    http://www.update.microsoft.com/

    Windows Update
    http://windowsupdate.microsoft.com/
