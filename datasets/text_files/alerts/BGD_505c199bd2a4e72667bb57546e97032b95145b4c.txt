Users are advised to install the applicable firmware updates for optimal protection. A hardcoded credential vulnerability was identified in the “zyfwp” user account in some Zyxel firewalls and AP controllers. The account was designed to deliver automatic firmware updates to connected access points through FTP. IMPACT As the zyfwp user has admin privileges, an attacker could completely compromise the confidentiality, integrity and availability of the device. SYSTEM AFFECTED Full list available on below URL: https://www.zyxel.com/support/CVE-2020-29583.shtml RECOMMENDATIONS Should update to the latest firmware as per vendor advisory.
