To exploit the vulnerability, an attacker would place malicious code in a specially crafted container image which, if an authenticated administrator imported (pulled), could cause a container management service utilizing the Host Compute Service Shim library to execute malicious code on the Windows host. Impact : A remote attacker could exploit this vulnerability to take control of an affected system. Mitigation : Apply an update. Please see the references or vendor advisory for more information.
