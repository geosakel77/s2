More Alerts Alert (SA05-193A) Microsoft Windows, Internet Explorer, and Word Vulnerabilities Original release date: July 12, 2005 Systems Affected Microsoft Windows Microsoft Office Microsoft Internet Explorer For more complete information, refer to the Microsoft Security Bulletin Summary for July, 2005 . Overview Microsoft has released updates that address critical vulnerabilities in
Windows, Internet Explorer, and Word. Description Microsoft Security Bulletins for July, 2005 address vulnerabilities in
Windows, Office, and Internet Explorer.

There are vulnerabilities in Microsoft Windows, Internet Explorer, and Word that may allow an attacker to take control of your computer or cause it to crash. 
For more technical information, see US-CERT Technical Alert TA05-193A . Resolution Apply Updates Microsoft has released security
updates for Windows, Internet Explorer, and Word. To obtain the updates,
visit the Microsoft 
Update web site. US-CERT also recommends enabling Automatic
Updates .
