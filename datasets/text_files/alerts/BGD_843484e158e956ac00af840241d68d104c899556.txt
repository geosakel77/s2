These updates resolve a critical vulnerability that could lead to arbitrary code execution in the context of the running ColdFusion service. Adobe is aware of a report that CVE-2019-7816 has been exploited in the wild. Impact : A remote attacker could exploit this vulnerability to take control of an affected system. Mitigation : Updates are available. Please see the references or vendor advisory for more information.
