More Alerts Alert (SA04-104A) Summary of Windows Security Updates for April 2004 Original release date: April 13, 2004 Systems Affected Systems running Microsoft Windows Overview There are multiple vulnerabilities in Microsoft Windows that could
allow attackers to take control of your computer. Description Microsoft has released Windows
Security Updates for April 2004 , which addresses multiple
vulnerabilities in the Microsoft Windows operating system. Three of
the four updates are considered critical, so users should apply the
updates as soon as possible. A technical description of these vulnerabilities is available from
US-CERT in TA04-104A and from Microsoft in MS04-011 , MS04-012 , MS04-013 ,
and MS04-014 . Resolution Apply a patch Follow the procedures outlined in Microsoft's Windows
Security Updates for April 2004 .
