More Alerts Alert (SA07-100A) Microsoft Updates for Multiple Vulnerabilities Original release date: April 10, 2007 Systems Affected Microsoft Windows Microsoft Internet Explorer Overview Microsoft has released updates to correct critical vulnerabilities in Microsoft Windows. Solution Install updates Microsoft has released updates to remedy critical vulnerabilities in Windows. To obtain these updates, visit the Microsoft Update web site. We also recommend enabling Automatic Updates . Description Vulnerabilities in Microsoft Windows may allow an attacker to access your computer, install and run malicious software on your computer, or cause it to crash. More technical information is available in US-CERT Technical Cyber Security Alert TA07-100A .
