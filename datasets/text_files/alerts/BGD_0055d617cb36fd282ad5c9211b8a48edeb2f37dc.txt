The attacker could also cause an affected system to reload, resulting in a denial of service (DoS) condition. The vulnerability is due to an incomplete input validation on user-controlled input in an HTTP request to the targeted device. An attacker could exploit this vulnerability by sending a crafted HTTP request to an affected system. Impact :  A successful exploit could allow the attacker to execute arbitrary code as the root user and gain full control of the affected system or cause it to reload, resulting in a DoS condition. Vulnerable Products RV132W ADSL2+ Wireless-N VPN Router RV134W VDSL2 Wireless-AC VPN Router Mitigation : Cisco has released software updates that address this vulnerability. Please see the references or vendor advisory for more information.
