This potentially allows attackers to exploit multiple attack vectors on a Drupal site, which could result in the site being compromised. This vulnerability is related to Drupal core – Highly critical – Remote Code Execution – SA-CORE-2018-002 . Both SA-CORE-2018-002 and this vulnerability are being exploited in the wild. Impact : An attacker could exploit this vulnerability to take control of an affected system. Mitigation : Apply an update. Please see the references or vendor advisory for more information.
