Impact : An attacker may leverage this issue to execute arbitrary script code in the browser of an unsuspecting user in the context of the affected site. This may allow the attacker to steal cookie-based authentication credentials and to launch other attacks. Versions prior to WP Statistics 12.0.5 are vulnerable. Mitigation : Updates are available. Please check specific vendor advisory for more information.
