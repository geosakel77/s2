Description : Mozilla has released a security update to address multiple vulnerabilities in Thunderbird. This attack only affects Windows operating systems. Other operating systems are unaffected. Impact : A remote attacker could exploit some of these vulnerabilities to take control of an affected system. Mitigation : Updates are available.
