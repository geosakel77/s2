I. Overview

  Microsoft has released its security bulletin for May, 2014. This
bulletin contains three (3) updates that are rated as "critical".
Remote attackers leveraging these vulnerabilities may be able to
execute arbitrary code.

  Details on the vulnerabilities can be found at the following URL:

    Microsoft Security Bulletin Summary for May 2014
    https://technet.microsoft.com/en-us/library/security/ms14-may

  [Security updates rated as "critical"]

    MS14-021
    Security Update for Internet Explorer (2965111)
    https://technet.microsoft.com/library/security/ms14-021
    
    MS14-029
    Security Update for Internet Explorer (2962482)
    https://technet.microsoft.com/library/security/ms14-029
    
    MS14-022
    Vulnerabilities in Microsoft SharePoint Server Could Allow Remote Code Execution (2952166) 
    https://technet.microsoft.com/library/security/ms14-022
    
  According to Microsoft, attacks leveraging MS14-021 and MS14-029
have been observed in the wild.


II. Solution

  Please apply the security update programs through Microsoft Update,
Windows Update, etc. as soon as possible.

    Microsoft Update
    http://www.update.microsoft.com/

    Windows Update
    http://windowsupdate.microsoft.com/

  * With the April 2014 update, Microsoft have ended the support for
    Windows XP and Office 2003. Concerns on security risk will rise
    henceforth, and therefore please consider updating to a newer OS
    and software.
