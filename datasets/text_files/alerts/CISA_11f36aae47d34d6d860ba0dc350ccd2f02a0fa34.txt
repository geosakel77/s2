More Alerts Alert (SA07-128A) Microsoft Updates for Multiple Vulnerabilities Original release date: May 08, 2007  | Last revised : May 09, 2007 Systems Affected Microsoft Windows Microsoft Internet Explorer Microsoft Office Overview Microsoft has released updates to correct critical vulnerabilities in Microsoft Windows, Internet Explorer, and Office. Solution Install updates Microsoft has released updates to remedy critical vulnerabilities in Microsoft Windows, Internet Explorer, and Office. To obtain these updates, visit the Microsoft Update web site. We also recommend enabling Automatic Updates . Description Vulnerabilities in Microsoft Windows, Internet Explorer, and Office may allow an attacker to access your computer, install and run malicious software on your computer, or cause it to crash. Microsoft Office for Apple OS X is also affected by some of these vulnerabilities. More technical information is available in US-CERT Technical Cyber Security Alert TA07-128A .
