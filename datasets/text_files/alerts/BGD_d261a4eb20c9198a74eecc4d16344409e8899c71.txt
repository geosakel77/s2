This flaw could be used by an unprivileged local attacker to, for example, create a new local administrator. The highest threat from this vulnerability is to data confidentiality and integrity as well as system availability. Impact: The vulnerability enables an unprivileged local user to get a root shell on the system. Mitigations: Updates are available.Please see the references or vendor advisory for more information.
