I. Overview

  Microsoft has released its security bulletin for October, 2014. This
bulletin contains three (3) updates that are rated as "critical".
Remote attackers leveraging these vulnerabilities may be able to
execute arbitrary code.

  Details on the vulnerabilities can be found at the following URL:

    Microsoft Security Bulletin Summary for October 2014
    https://technet.microsoft.com/en-us/library/security/ms14-oct

  [Security updates rated as "critical"]

    MS14-056
    Cumulative Security Update for Internet Explorer (2987107)
    https://technet.microsoft.com/en-us/library/security/ms14-056

    MS14-057
    Vulnerabilities in .NET Framework Could Allow Remote Code Execution (3000414)
    https://technet.microsoft.com/en-us/library/security/ms14-057

    MS14-058
    Vulnerability in Kernel-Mode Driver Could Allow Remote Code Execution (3000061)
    https://technet.microsoft.com/en-us/library/security/ms14-058


II. Solution

  Please apply the security update programs through Microsoft Update,
Windows Update, etc. as soon as possible.

    Microsoft Update
    http://www.update.microsoft.com/

    Windows Update
    http://windowsupdate.microsoft.com/

  * With the April 2014 update, Microsoft has ended the support for
    Windows XP and Office 2003. Concerns on security risk will rise
    henceforth, and therefore please consider updating to a newer OS
    and software.
