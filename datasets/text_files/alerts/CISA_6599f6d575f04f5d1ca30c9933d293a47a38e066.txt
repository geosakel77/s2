Description As part of the Microsoft Security Bulletin Summary for March 2009 , Microsoft released updates to address vulnerabilities that affect Microsoft Windows and Windows Server. Impact A remote, unauthenticated attacker could gain elevated privileges, poison the DNS cache, execute arbitrary code, or cause a vulnerable application to crash. Solution Microsoft has provided updates for these vulnerabilities in the Microsoft Security Bulletin Summary for March 2009 . The security bulletin describes any known issues related to the updates. Administrators are encouraged to note these issues and test for any potentially adverse effects. Administrators should consider using an automated update distribution system such as Windows Server Update Services ( WSUS ).
