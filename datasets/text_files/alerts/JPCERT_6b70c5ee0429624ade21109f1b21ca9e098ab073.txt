         Microsoft Releases October 2022 Security Updates

       https://www.jpcert.or.jp/english/at/2022/at220028.html


I. Overview
Microsoft has released October 2022 Security Updates to address the
vulnerabilities in their products. Remote attackers leveraging
these vulnerabilities may be able to execute arbitrary code. It is
recommended to check the information provided by Microsoft and
apply the updates.

    Microsoft Corporation
    October 2022 Security Updates
    https://msrc.microsoft.com/update-guide/en-us/releaseNote/2022-Oct

    Microsoft Corporation
    Microsoft Security Updates for October 2022 (Monthly) (Japanese)
    https://msrc-blog.microsoft.com/2022/10/11/202210-security-updates/

According to Microsoft, among these vulnerabilities, the following
vulnerability has been confirmed to be exploited in the wild. Please
consider applying the security update programs by referring to the
information provided by Microsoft.

    CVE-2022-41033
    Windows COM+ Event System Service Elevation of Privilege Vulnerability
    https://msrc.microsoft.com/update-guide/ja-JP/vulnerability/CVE-2022-41033


II. Solution
Please apply the security update programs through Microsoft Update,
Windows Update, etc. as soon as possible.

    Microsoft Update Catalog
    https://www.catalog.update.microsoft.com/

    Windows Update: FAQ
    https://support.microsoft.com/en-us/help/12373/windows-update-faq


III. Related information
On September 30, 2022 (local time), Microsoft released a blog and
guidance regarding Microsoft Exchange Server zero-day vulnerabilities (CVE-2022-41040, CVE-2022-41082). The blog explains the attack Microsoft
observed in August, and the guidance provided information on mitigation measures and detection queries until the patch is released.

As of October 12, 2022, the update to fix the vulnerabilities has not
been released. The users of the product are recommended to apply
mitigation measures while keeping an eye on the information published
by Microsoft and applying the update immediately once it is available.

    Microsoft Corporation
    Customer Guidance for Reported Zero-day Vulnerabilities in Microsoft Exchange Server
    https://msrc-blog.microsoft.com/2022/09/29/customer-guidance-for-reported-zero-day-vulnerabilities-in-microsoft-exchange-server/

    Microsoft Security Threat Intelligence
