It’s possible for the local attacker to create symbolic links before the files are to be created and possibly escalating the privileges with the ownership change. Impact : A local attacker may exploit this issue to gain elevated root privileges on the affected system. Mitigation : Updates are available. Please check specific vendor advisory for more information.
